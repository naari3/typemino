# ゲームモード一覧

すべてのゲームモードはstandardを継承している

- standard
  - 常に同じ設定値のもと無限にプレイできるモード
- master3
  - ミノの設置やラインの消去によって上がるレベルを999に到達させるのが目的のモード
  - レベルが上がるに連れて各種設定値が厳しくなっていく
- dig
  - ゲーム開始時に画面上に存在するミノを消去し、一番下まで到達させるのが目的のモード
- dig challenge
  - 一定時間ごとに一行ずつミノがせり上がってくる
  - いつまで生存していられるかを競うのが目的のモード
